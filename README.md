# Projeto Dados dos Atletas

Este projeto foi desenvolvido como parte de uma 2° atividade de certificação em JavaScript. 

# Descrição do Projeto
O objetivo é criar uma aplicação capaz de receber os dados de um atleta e calcular:
- Categoria do atleta (por idade)
- IMC (Índice de Massa Corporal)
- Média válida das notas (desconsiderando a maior e a menor)

## Como executar o projeto

Pelo computador (Node.js)
